# CampusCart:The ultimate campus marketplace

A full-featured eCommerce platform built with the MERN stack (MongoDB, Express, React, Node.js) and Redux Toolkit. Includes shopping cart, checkout, product reviews, PayPal payments, and an admin area for managing products, users, and orders.

## Features

✅ Full shopping cart & checkout flow  
✅ Product reviews and ratings  
✅ Product search and pagination  
✅ User profiles with order history  
✅ Admin product, user, and order management  
✅ PayPal / credit card payments  
✅ Responsive design

## Technologies

- **Frontend:** React, Redux Toolkit, React Bootstrap
- **Backend:** Node.js, Express
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT
- **Payments:** PayPal REST API
